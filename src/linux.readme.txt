# To add IP tools
To temporarily add additional tooling to the container 

Connect to the console (which is root) via the Containers panel.
```
	apt update
```

Then either;

```
	apt-get install iputils-ping
```

and/or;

```
	apt-get install net-tools 
```

# for privileged access

```
apt install sudo
```

# For editing files
apt install nano
